# Instruction for MD work

## Text selection

To make italics font, add *  *around the text*. 
To make bold font, add **    **around the text**.
 To make italics+bold font, add ***  ***around the text***.   
 To make crossed font, add double tilda (~~)   ~~around the text~~.
